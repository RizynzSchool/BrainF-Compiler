# BrainF-Compiler
BrainF Compiler is a python made compiler for BrainF**k.

# Getting Started:
Clone Repo: ```git clone --recursive https://github.com/RizynzSchool/BrainF-Compiler```<br />
Compile & Run: ```cd BrainF-Compiler/src; bash run.bash```<br />
